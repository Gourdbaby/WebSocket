# WebSocket
Websocket example

### How to use
  * #### use commend
      `git clone https://github.com/huluwa123/WebSocket.git`   
      `cd WebSocket`   
      `npm i`   
      `npm run start`   
      `port 3000, http://localhost:3000`
  * #### Introduction
      使用了node的ws库，在服务端准备和客户端建立websocket连接，   
      简单的做了一个收发消息的功能   
      注意多台电脑测试时，首先保证IP互通，其次需要把views/index.html文件的ws链接地址改成你电脑的IP
